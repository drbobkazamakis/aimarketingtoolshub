Suggest a tool, flag a reclassification, or push back on a rating. The directory gets better when readers correct it.

What we read carefully

  • Tool suggestions. If we’ve missed a tool that’s genuinely in the AI marketing category, send it. Include a link to the tool’s technical documentation if it has any — that’s the most useful single signal.
  • Reclassification arguments. If you think a tool is wrongly tagged Real AI, Hybrid, or Marketing AI, walk us through your reasoning. Specific evidence beats general disagreement.
  • Vendor corrections. If you work at a vendor and we’ve misrepresented your product, write in with what we got wrong — ideally with a pointer to the documentation that contradicts our reading. Corrections that hold up get reflected in the next quarterly update.
  • Methodology suggestions. The diagnostic questions evolve. If you have one we should be asking that we’re not, send it.

What we don’t do

  • Sponsored placements. The directory has no paid slots, no sponsored reviews, no “featured” tiers. Don’t pitch them.
  • Affiliate partnerships in exchange for ranking. Tool order is editorial. We don’t adjust ranking for commercial reasons.
  • Backlink exchanges. We don’t trade links. Editorial decisions only.
